API ЮMoney для сбора денег
В ЮMoney есть специальный сервис для коллективного сбора денег. Этот инструмент подойдет всем, кому нужно получать деньги для личных целей: например, собирать на совместные покупки, общие подарки или свой проект.
С помощью этого API вы можете сделать нестандартную форму для сбора денег и настроить для нее HTTP-уведомления.
Возможности
Вы можете собрать виджет или кнопку в конструкторе и разместить их на сайте или в блоге.
Если вам нужны дополнительные функции или собственный дизайн, создайте свою форму по этому API. С его помощью можно:
  • принимать переводы в кошелек ЮMoney — с банковских карт и из кошельков ЮMoney,
  • получать уведомления об этих переводах.
Как использовать
Виджет или кнопка в конструкторе
  1. Соберите виджет или кнопку в конструкторе.
  2. Разместите HTML-код на своем сайте.
  3. Подключите сообщения о переводах: по email или с помощью HTTP-уведомлений.
Информация о созданном сборе будет отображаться на странице ваших сборов.
Нестандартная форма
  1. Соберите форму по API.
  2. Включите HTTP-уведомления.
  3. Реализуйте прием уведомлений.
Подробная инструкция
Информация о сборах с использованием нестандартных форм не отображается на странице сборов.
